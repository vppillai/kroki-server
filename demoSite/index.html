Zoom & Pan Controls

                
            

            

                
Mouse Controls:

                
    
                    
  • Mouse Wheel: Zoom in/out
    • 
                    
  • Click & Drag: Pan around the diagram
    • 
                    
  • Double-click: Reset zoom to fit
    • 
                

                
Touch Controls (Mobile):

                
    
                    
  • Pinch: Zoom in/out
    • 
                    
  • Single finger drag: Pan around
    • 
                

                
Keyboard Shortcuts:

                
    
                    
  • Ctrl/Cmd + Plus (+): Zoom in
    • 
                    
  • Ctrl/Cmd + Minus (-): Zoom out
    • 
                    
  • Ctrl/Cmd + 0: Fit to screen
    • 
                    
  • F key: Enter fullscreen mode
    • 
                    
  • Escape key: Exit fullscreen mode
    • 
                

                
Button Controls:

                
    
                    
  • + / - buttons: Zoom in/out from center
    • 
                    
  • ⌂ (Fit to Screen) button: Reset zoom to fit diagram
    • 
                    
  • ⤡ (Fullscreen) button: Toggle maximum screen view
    • 
                

                
Fullscreen Mode:

                
    
                    
  • Hides the code panel for maximum diagram viewing space
    • 
                    
  • Use the ⤡ button or press F to enter
    • 
                    
  • Press Escape or click ✕ to exit
    • 
                    
  • All zoom and pan controls remain available
    • 
                

                
File Operations:

                
    
                    
  • Ctrl/Cmd + N: New file
    • 
                    
  • Ctrl/Cmd + O: Open file from disk
    • 
                    
  • Ctrl/Cmd + S: Save current file
    • 
                    
  • Ctrl/Cmd + Shift + S: Save As new file
    • 
                    
  • Supports various diagram formats (.puml, .mmd, .dot, .d2, etc.)
    • 
                    
  • Auto-detects diagram type from file content/extension
    • 
                    
  • Shows file status and warns about unsaved changes
    •
